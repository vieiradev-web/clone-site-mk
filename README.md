# 🚀 MK Telecom - Landing Page

Uma landing page moderna, de alta performance e totalmente responsiva desenvolvida para a **MK Telecom**, focada na conversão de novos clientes para planos de internet de fibra óptica pura e telefonia móvel (MK Móvel).


## 🎯 Funcionalidades Principais

*   **Consulta Interativa de CEP:** Sistema em JavaScript que valida a estrutura do CEP (formato `00000-000`) em tempo real e simula a viabilidade técnica de cobertura na região do cliente.
*   **Redirecionamento Inteligente para WhatsApp:** Links de contratação parametrizados que enviam mensagens automáticas personalizadas com o plano escolhido ou o CEP consultado diretamente para a equipe de vendas.
*   **Seção MK Móvel Dedicada:** Apresentação do novo plano pré-pago de 30GB com elementos visuais modernos que imitam a interface de um smartphone.
*   **Ícones Dinâmicos:** Integração fluida com a biblioteca **Lucide Icons** para manter a interface limpa e intuitiva.
*   **Botão Flutuante de Suporte:** Canal direto via WhatsApp fixado na tela para facilitar o contato do usuário a qualquer momento.
*   **Speed Test Integrado:** Atalho direto no cabeçalho para que os usuários possam testar a velocidade da sua conexão atual de forma prática.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando tecnologias web fundamentais, priorizando carregamento rápido e sem dependências pesadas:

*   **HTML5:** Estruturação semântica e acessível.
*   **CSS3:** Estilização moderna com variáveis globais, layouts em Grid/Flexbox, efeitos de brilho (*glow neon*) e responsividade total para dispositivos móveis.
*   **JavaScript (Vanilla):** Lógica de máscaras de entrada, manipulação dinâmica do DOM para mensagens de feedback e inicialização assíncrona de ícones.
*   **Lucide Icons:** Biblioteca de ícones em SVG leve e escalável.
